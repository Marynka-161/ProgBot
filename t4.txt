Змінні

  Змінні зберігають дані для обробки.
  Змінній присвоюється ім'я ( ідентифікатор ), area наприклад, age, height тощо. Ім'я ідентифікує кожну змінну, привласнюючи їй значення і отримуючи збережене значення.

  Змінні мають типи. Деякі приклади:
- int : для цілих чисел (цілих чисел), таких як 123 і -456
- double : для чисел з плаваючою комою і дробовими частинами фіксованих або наукових позначень, таких як 3.1416, -55.66.
- String : для таких текстів, як" Привіт "або" Доброго ранку!". Текстові рядки, які знаходяться в подвійних лапках.

  Ви можете оголосити змінну певного типу та присвоїти їй значення. 
  Приклад:
String name = "Андрій"; 

це створює змінну з ім'ям name типу String і присвоює їй значення "Андрій".

Приклади оголошень змінних:
class MyClass { 
  public static  void main (String [ ] args) { 
    String name = "Андрій"; 
    int age = 42; 
    double score =15.9; 
    char group = 'Z';
  } 
} 

char означає character(символ) і містить один символ.

  Іншим типом є тип boolean, який має тільки два можливих значення: true і false .
  Цей тип даних використовується для простих умов істинна/хиба.
  Наприклад:
boolean online = true;
