Оператор инкремента

  Оператор инкремента або декремента забезпечує більш зручний і компактний спосіб збільшення або зменшення значення змінної на одиницю.
  Наприклад, оператор x=x+1 можна спростити до ++x;
  Приклад:
int test = 5; 
++test; / / test = 6 

  Оператор декремента (--) використовується для зменшення значення змінної на одиницю.
int test = 5; 
--test; / / test = 4 

Префікс та постфікс

  Дві форми, префікс і постфікс, можуть використовуватися як з операторами инкремента, так і з операторами декремента.
  З префиксной формою оператор з'являється перед операндом, в той час як в постфиксной формі оператор з'являється після операнда. Нижче наводиться пояснення того, як працюють дві форми:
  Префікс: збільшує значення змінної і використовує нове значення у вираженні.
  Приклад:
int x = 34; 
int y = ++x ; / / y - це 35 

  Значення x спочатку збільшується до 35, а потім присвоюється y, тому значення як x, так і y тепер дорівнюють 35.
  Постфікс: значення змінної спочатку використовується у виразі, а потім збільшується.
  Приклад:
int x = 34; 
int y = x++; / / y - 34 

Спочатку x присвоюється y,а потім збільшується на одиницю. Таким чином, x стає 35, а y присвоюється значення 34.

Оператор присвоювання

Ви вже знайомі з оператором присвоювання (=), який присвоює значення змінній.

int value = 5; / / цей вираз присвоює значення 5 змінній value типу int

  Java надає ряд операторів присвоювання, щоб спростити написання коду.

  Додавання та присвоєння (+=):
int num1 = 4; 
int num2 = 8; 
num2 += num1 ; / / num2 = num2 + num1

  Віднімання та присвоєння (-=):
int num1 = 4; 
int num2 = 8; 
num2 -= num1; / / num2 = num2-num1
