Математичні Оператори

  Java надає багатий набір операторів для використання в управлінні змінними. Значення, яке використовується з обох сторін оператора називається операндом.
  Наприклад, у наведеному нижче вираженні числа 6 і 3 є операндами оператора plus: 
int x = 6 + 3; 

Арифметичні оператори Java 

+ додавання
- віднімання
* множення
/ ділення
% ділення по модулю

Додавання

  Оператор + складає разом два значення, такі як дві константи, константу і змінну, або змінну і змінну. Ось кілька прикладів додавання:
int sum1 = 50 + 10; 
int sum2 = sum1 + 66; 
int sum3 = sum2 + sum2;

Віднімання

  Оператор -  віднімає одне значення від іншого.
int sum1 = 1000-10; 
int sum 2 = sum1-5; 
int sum3 = sum1 - sum 2; 

Множення

  Оператор * примножує два значення.
int sum1 = 1000 * 2; 
int sum 2 = sum1 * 10; 
int sum3 = sum1 * sum 2;

Розподіл

 Оператор / ділить одне значення на інше.
int sum1 = 1000 / 5; 
int sum 2 = sum1 / 2; 
int sum3 = sum1 / sum 2; 

Ділення по модулю

  Математична операція ділення по модулю (або залишок від ділення) виконує цілочисельне ділення одного значення на друге і повертає залишок від цього поділу.
  Оператором для операції ділення по модулю є символ відсотка ( % ).
  Приклад:
int value = 23; 
int res = значення % 6; / / res = 5
