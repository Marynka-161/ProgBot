Ласкаво просимо на Java!

  Java - це сучасна мова програмування високого рівня, розроблена на початку 1990-х років компанією Sun Microsystems і в даний час належить Oracle.

  Java не залежить від платформи, що означає, що вам потрібно лише написати програму один раз, щоб мати можливість запускати її на декількох різних платформах!
  Java є портативною, надійною і динамічною мовою програмування, з можливістю відповідати потребам практично будь-якому типу програм.

Java

  Понад 3 мільярдів пристроїв працюють на Java.
  Java використовується для розробки додатків для Google Android OS, різних настільних додатків, таких як медіа-плеєри, антивірусні програми, веб-додатки, корпоративні програми (тобто банківські) та багато інших!
